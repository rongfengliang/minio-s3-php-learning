# minio s3 php learning

> user aws s3 sdk 


## how to Running

* create .env file

> replace value with your s3 config


```code
time_zone=Asia/Shanghai
regin=default
endpoint=<endpoint>
use_path_style_endpoint=true
access_key=<access_key>
access_secret=<access_secret>
s3_bucket=<s3_bucket>
test_key=<just for test key >
```


* runing 

```code
composer install
php demo.php
```
